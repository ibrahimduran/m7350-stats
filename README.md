# m7350-stats

Simple program for displaying statistics from TP-Link M7350 in system tray. Uses HTTP service provided for the admin panel. Mainly developed for Windows 10.

## Environment

    Brand: TP-Link
    Type: LTE Modem
    Model: M7350
    Firmware Version: 6.2.2 Build 200821 Rel.1004n
    OS: Windows 10
